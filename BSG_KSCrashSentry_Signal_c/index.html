bsg_kssighndl_i_handleSignal

void bsg_kssighndl_i_handleSignal(
    int sigNum,
    siginfo_t *signalInfo, 
    void *userContext)
Parameters
sigNum

The signal that was raised.

signalInfo

Information about the signal.

userContext

Other contextual information.

Discussion

Our custom signal handler. Restore the default signal handlers, record the signal information, and write a crash report. Once we're done, re-raise the signal and let the default handlers deal with it.
