bsg_ksbt_backtracePthread

int bsg_ksbt_backtracePthread(
    pthread_t thread,
    uintptr_t *backtraceBuffer, 
    int maxEntries);
Parameters
thread

The thread to generate a backtrace for.

backtraceBuffer

A buffer to hold the backtrace.

maxEntries

The maximum number of trace entries to generate (must not be larger than backtraceBuffer can hold).

Return Value

The number of backtrace entries generated.

Discussion

Generate a backtrace on the specified posix thread (async-safe).

bsg_ksbt_symbolicate

void bsg_ksbt_symbolicate(
    const uintptr_t *backtraceBuffer, 
    Dl_info *symbolsBuffer,
    int numEntries, 
    int skippedEntries);
Parameters
backtraceBuffer

A backtrace generated by one of the bactrace_xx() methods.

symbolsBuffer

A buffer to hold the symbolicated backtrace.

numEntries

The number of entries to examine.

skippedEntries

The number of entries skipped from the start of this backtrace.

Discussion

Symbolicate a backtrace (async-safe).

This will query the linker and fill out as much information as it can about a stack trace. Any entries where it could not find info will have Dl_info members set to NULL.
