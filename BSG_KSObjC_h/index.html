bsg_ksobjc_getDescription

size_t bsg_ksobjc_getDescription(
    void *object,
    char *buffer, 
    size_t bufferLength);
Parameters
object

the object to generate a description for.

buffer

The buffer to copy into.

bufferLength

The length of the buffer.

Return Value

the number of bytes copied (not including null terminator).

Discussion

Generate a description of an object.

For known common object classes it will print extra information. For all other objects, it will print a standard <SomeClass: 0x12345678>

For containers, it will only print the first object in the container.

buffer will be null terminated unless bufferLength is 0. If the string doesn't fit, it will be truncated.

bsg_ksobjc_objectType

BSG_KSObjCType bsg_ksobjc_objectType(
    const void *objectOrClassPtr);
Parameters
objectOrClassPtr

Pointer to something that may be an object or class.

Return Value

The type of object, or BSG_KSObjCTypeNone if it was not an object or was inaccessible.

Discussion

Query a pointer to see what kind of object it points to. If the pointer points to a class, this method will verify that its basic class data and ivars are valid, If the pointer points to an object, it will verify the object data (if recognized as a common class), and the isa's basic class info (everything except ivars).

Warning: In order to ensure that an object is both valid and accessible, always call this method on an object or class pointer (including those returned by bsg_ksobjc_isaPointer() and bsg_ksobjc_superclass()) BEFORE calling any other function in this module.
